# Microsoft Edge Add-ons 发布资料

## 名称

DDYS Edge Extension

## 简短描述

低端影视 DDYS 的 Edge 扩展：搜索、最新、热门、日历、收藏、订阅提醒和资源入口。

## 详细描述

DDYS Edge Extension 为低端影视用户提供浏览器内的快捷入口。用户可以在工具栏弹窗、Edge 侧边栏和新标签页中搜索影片、查看最新更新、热门内容、日历、影片详情和资源入口；也可以收藏、稍后看、记录历史、添加本地备注、订阅关键词并接收 Edge 通知。

扩展支持右键搜索选中文字、地址栏 `ddys` 关键词搜索、页面选区快捷搜索、自定义 API Base、缓存、诊断、数据导入导出。

## 分类

Entertainment

## 隐私摘要

扩展不出售、不共享、不上传用户收藏、备注、观看记录和订阅。数据保存在浏览器本地。网络请求仅发送到用户配置的 DDYS API Base。

## 发布包

使用 `ddys-edge-extension-v0.1.0.zip` 上传到 Microsoft Partner Center 的 Microsoft Edge Add-ons 项目。
