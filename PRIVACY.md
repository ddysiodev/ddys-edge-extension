# 隐私说明

DDYS Edge Extension 不采集、出售或上传用户的浏览历史、收藏、备注、订阅和设置。

扩展会在以下场景发起网络请求：

- 用户搜索、打开最新、热门、日历、影片详情和资源入口时，请求用户配置的 DDYS API Base。
- 后台订阅提醒开启时，按用户设置的间隔请求 DDYS API，检查订阅关键词是否有新结果。
- 用户使用自定义 API Base 时，请求会发送到用户填写的服务地址。

以下数据保存在浏览器本地 `chrome.storage.local`：

- API Base、站点 Base、缓存时间、刷新间隔、打开方式等设置。
- 收藏、稍后看、观看记录、本地备注。
- 关键词订阅及最近命中的结果标识。
- API 短期缓存和诊断结果。

Microsoft Edge 的 Chromium 扩展运行时仍使用 `chrome.*` API 命名空间。扩展不包含远程脚本，不会从第三方地址加载可执行代码。
